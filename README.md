鴻蒙 HarmonyOS 粵拼輸入法
======

（開發中，未用得。）

採用 [香港語言學學會粵語拼音方案](https://jyutping.org/jyutping) (粵拼 / Jyutping)，兼容各種習慣拼寫串法。  
候選詞會標注對應嘅粵拼。  
支持簡、繁體漢字及各種字形標準。  
可以用倉頡、速成、筆畫、普通話拼音、拆字等反查粵語拼音。

另有 Android、iOS、iPadOS 及 macOS 版: https://jyutping.app
